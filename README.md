# Docs

## Introduction

This repository holds the technical documentation.

## Getting Started

```shell
python3 -m venv venv
make install_dev
```

## Run Locally

Start a simple HTTP server:

```shell
python3 -m http.server
```

Navigate to `localhost:8000/docs`.

## Export to PDF

Consider [this project](https://github.com/kernoeb/docker-docsify-pdf).
